# Hello 👋

Je suis **Salim Benfarhat**, développeur fullstack et partenaire technique — *Fullstack Product Delivery*.

Je conçois et livre des produits digitaux fonctionnels, de l'idée au déploiement. Mon approche est orientée cadrage, livraison rapide, autonomie et qualité.

🚀 **Stack principale :** 
- **Frontend** (Next.js, React, TypeScript, Tailwind CSS) 
- **Backend** (Node.js, Express, NestJS) 
- **Database** (PostgreSQL, Prisma) 
- **Authentication** (Better Auth)  
- **Testing** (Vitest, Playwright)  
- **DevOps** (Docker) 
- **APIs** (REST) 
- **AI** (OpenRouter, OpenAI, Claude, Gemini)
